## Get started
- Clone repository
  ```bash
  git clone https://github.com/atlp-rwanda/e-commerce-ninjas-fe
  ```
- Install dependencies
  ```bash
  npm install
  ```
- Run App - it will run on http://localhost:5000/
  ```bash
  npm run dev
  ```


## StoryBook
To run storybook for documentation, use the command 
```bash
npm run storybook
```
